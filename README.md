# Playwright Practice

Учебный проект по автоматизации тестирования на **JavaScript** с использованием **Playwright**, выполненный в рамках стажировки в **Innowise**.

Проект создан для практики написания e2e-тестов, работы с тестовым фреймворком Playwright и освоения базовых подходов к автоматизации UI-тестирования.

---

## Стек технологий

- JavaScript
- Node.js
- Playwright
- Git / GitHub

---

## Структура проекта

- `tests/` — автотесты
- `playwright.config.js` — конфигурация Playwright
- `test-results/` — результаты запусков тестов (игнорируются в git)
- `playwright-report/` — HTML-отчёты (игнорируются в git)

---

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/dizdonchik-png/playwright-practice.git
cd playwright-practice
```

### 2. Установить зависимости
```bash
npm install
```

### 3. Запуск тестов
```bash
npx playwright test
```

### 4. Просмотр HTML-отчёта
```bash
npx playwright show-report
```

## Цели проекта
- Изучение Playwright и его возможностей
- Практика написания e2e-тестов
- Работа с локаторами, асинхронностью, ожиданиями
- Настройка тестового проекта и конфигурации
- Работа с Git и GitHub
